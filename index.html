1. 기본 학습

한 화면에 영어 문장 하나가 나옵니다. 단어를 눌러 사전을 열고, 🔊 버튼으로 발음을 들으세요.

2. 번역과 설명 보기

카드를 누르거나 Space 키를 누르면 한국어 번역과 구조 설명을 볼 수 있습니다. 항상 보려면 아래 설정에서 켜두면 됩니다.

3. 복습하기

중요한 문장은 ★로 저장하세요. 하단의 ★ 즐겨찾기 탭에서 저장한 문장만 빠르게 다시 볼 수 있습니다.

4. 새 영어 자료 넣기

HTML/JSON/TXT 파일을 업로드하거나, 웹주소를 넣어 핵심 영어 문단을 추출하거나, 영문 텍스트를 직접 붙여넣을 수 있습니다.

5. 자동 보강

번역이 없는 영어 자료는 자동으로 한국어 번역과 단어 설명을 보강합니다. 결과가 마음에 들지 않으면 텍스트칸에서 수정한 뒤 저장하세요.

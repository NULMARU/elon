# 영한번역 사용자 매뉴얼

영한번역은 영어 원문을 한 문장씩 나누어 듣고, 번역을 확인하고, 핵심 단어를 복습하는 스마트폰용 영어 학습 앱입니다.

## 1. 기본 학습 방법

1. 화면 중앙의 영어 문장을 읽습니다.
2. `🔊` 버튼을 눌러 발음을 듣습니다.
3. 문장을 누르거나 `Space` 키를 누르면 한국어 번역과 설명이 보입니다.
4. `다음` 버튼으로 다음 문장으로 이동합니다.

## 2. 단어 확인하기

영어 단어를 누르면 작은 사전 팝업이 열립니다.

- `Naver 사전`: 한국어 뜻을 빠르게 확인할 때 사용합니다.
- `Cambridge`: 영어식 설명과 예문을 확인할 때 사용합니다.

## 3. 즐겨찾기 복습

중요하거나 어려운 문장은 `★` 버튼으로 저장합니다.

하단의 `★ 즐겨찾기` 탭을 누르면 저장한 문장만 모아서 복습할 수 있습니다.

## 4. 새 영어 자료 가져오기

설정에서 `학습용 새 영어 파일 가져오기`를 사용합니다.

- `HTML / JSON / TXT 파일 가져오기`: 저장된 파일을 앱에 넣습니다.
- `웹주소에서 핵심 영어 추출`: 웹주소를 넣으면 핵심 영어 문단을 추출해 텍스트칸에 넣습니다.
- `영문 텍스트 직접 붙여넣기`: 기사, 대본, 가사, 문단을 직접 붙여넣습니다.

텍스트가 준비되면 `영문 텍스트로 카드 만들기`를 누릅니다.

## 5. 자동 번역과 단어 설명

번역이 없는 영어 자료는 앱이 자동으로 한국어 번역과 단어 설명을 보강합니다.

자동 결과는 완벽하지 않을 수 있으므로, 중요한 자료는 저장하기 전에 텍스트칸에서 한번 확인하는 것이 좋습니다.

## 6. 보기와 듣기 설정

설정 화면에서 아래 항목을 바꿀 수 있습니다.

- 앱 테마: `Dark`, `Light`, `Sepia`
- 글자 크기
- 한 페이지 분량
- 영어 듣기 재생 배속
- 영어 발음 억양/목소리
- 새 문장 진입 시 자동 재생
- 번역 및 상세 설명 항상 노출

## 7. 단축키

- `Space`: 번역 보기
- `← / →`: 이전/다음 문장
- `Enter`: 발음 듣기

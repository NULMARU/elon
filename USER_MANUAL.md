# 영한번역 사용자 매뉴얼

영한번역은 영어 원문을 학습 카드로 나누어 듣고, 번역을 확인하고, 핵심 단어를 복습하는 스마트폰용 영어 학습 앱입니다.

## 1. 기본 학습 방법

1. 화면 중앙의 영어 문장을 읽습니다.
2. `🔊` 버튼을 눌러 발음을 듣습니다. `🐢` 버튼은 느리게(0.7x) 들려줍니다.
3. 문장을 누르거나 `Space` 키를 누르면 한국어 번역과 설명이 보입니다.
4. `다음` 버튼으로 다음 문장으로 이동합니다.

## 2. 단어 확인하기

주 화면 상단의 `A- / A+` 버튼으로 영어 문장과 번역 설명의 글자 크기를 바로 조절할 수 있습니다.

영어 단어를 누르면 작은 사전 팝업이 열립니다.

- `✨ AI 문맥 뜻 보기`: 이 문장 안에서의 정확한 뜻을 AI가 설명해 줍니다. (Gemini API 키 필요)
- `Naver 사전`: 한국어 뜻을 빠르게 확인할 때 사용합니다.
- `Cambridge`: 영어식 설명과 예문을 확인할 때 사용합니다.

## 3. 즐겨찾기 복습

중요하거나 어려운 문장은 `★` 버튼으로 저장합니다.

하단의 `★ 즐겨찾기` 탭을 누르면 저장한 문장만 모아서 복습할 수 있습니다.

## 4. 새 영어 자료 가져오기 (상단 ➕ 버튼)

워크플로: **웹주소/유튜브 주소 또는 영문 텍스트 입력 → 영문 추출 → AI 학습 카드 생성 → 카드 독해 + 듣기**

1. **1단계 — 주소 입력**: 유튜브 주소를 넣으면 youtubetotext.ai로 자막을 추출하고, 일반 웹주소를 넣으면 본문을 추출합니다.
2. **2단계 — 텍스트 확인**: 추출된 영문이 텍스트칸에 들어옵니다. 필요하면 수정하고, 영문을 직접 붙여넣어도 됩니다.
3. **`🤖 AI 학습 카드 만들기`**: Gemini가 분량 설정에 맞게 카드를 나누고, 자연스러운 번역·문맥 단어 뜻·직독직해 해설·필수 암기 표현까지 만들어 줍니다.

HTML / JSON / TXT 파일 업로드도 같은 패널에서 할 수 있습니다.

Gemini API 키가 없으면 기존 방식(자동 번역 + 규칙 기반 설명)으로 카드를 만듭니다.

## 5. 관심분야 AI 추천 자료

➕ 패널 하단에서 관심분야(예: 우주, AI, 테슬라)를 입력하고 `오늘의 추천 학습 글 생성`을 누르면, 내 관심분야와 영어 수준에 맞는 독해 지문을 AI가 새로 써서 카드로 만들어 줍니다. 최근 추천과 겹치지 않는 주제를 자동으로 고릅니다.

## 6. API 키 설정 (설정 ⚙️ → AI / API 키 설정)

- **Gemini API 키**: [aistudio.google.com](https://aistudio.google.com) → `Get API Key`에서 무료 발급. AI 카드 생성·문맥 단어 뜻·추천 자료 기능이 켜집니다.
- **youtubetotext.ai API 키**: youtubetotext.ai 대시보드에서 확인. 유튜브 자막 추출에 사용됩니다.
- 키는 이 기기 브라우저에만 저장되며 외부로 전송되지 않습니다 (해당 API 호출 제외).

## 7. 보기와 듣기 설정

설정 화면에서 아래 항목을 바꿀 수 있습니다.

- 앱 테마: `Dark`, `Light`, `Sepia`
- 글자 크기 / 한 페이지 분량
- 영어 듣기 재생 배속
- 영어 발음 억양/목소리 — 기기에서 가장 자연스러운 음성이 `★` 표시와 함께 자동 선택됩니다. Android는 `Google` 음성, iPhone은 `Siri/Enhanced` 음성이 가장 좋습니다.
- 새 문장 진입 시 자동 재생 / 번역 및 상세 설명 항상 노출
- 내 영어 수준 (초급/중급/고급 — AI 카드와 추천에 반영)

> **Android 팁**: 설정 → 일반 → 텍스트 음성 변환(TTS)에서 Google 음성 엔진의 영어(미국) 고품질 음성 데이터를 다운로드해 두면 오프라인에서도 발음이 좋아집니다.

## 8. 단축키

- `Space`: 번역 보기
- `← / →`: 이전/다음 문장
- `Enter`: 발음 듣기
